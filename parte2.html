Var, const e tipos

Explicação

Var e const nos permitem guardar valores para usarmos em nossa aplicação

Tipos são

String = Texto

Number = números

Boolean = Verdadeiro ou Falso, só isso

Arrays = É uma estrutura de dado para armazenar uma coleção de valores, sendo eles de qualquer tipo.

Objects = Conjunto de atributos aninhados a uma variável denomina-se um objeto.

Funções = Vamos falar mais delas na frente


Para Praticar

Crie várias var e const usando todos os tipos e utilizando o console.log(), não esqueça de apertar f12
